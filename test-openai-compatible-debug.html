常见问题解决方案

!
错误：Unexpected token '<', "<!doctype "... is not valid JSON
原因：服务器返回HTML页面而不是JSON。检查URL是否正确，服务是否运行。
!
错误：HTTP 404 Not Found
原因：API端点不存在。确认URL路径是否正确（通常是 /v1/chat/completions）。
!
错误：HTTP 401 Unauthorized
原因：API密钥无效或缺失。检查密钥是否正确配置。
!
连接超时
原因：服务响应慢或不可达。检查网络连接和服务状态。
